프로젝트 설명

간단한 ToDoList Web Application
Node.js 서버개발자 채용공고의 자격요건을 충족하기 위해 시작한 프로젝트
프론트엔드는 빠른 개발을 위해 React보다 익숙한 Next.js를 사용. Redux-Saga를 사용하여 상태관리하고, Mui를 사용하여 디자인
백엔드는 Node.js 와 MongoDB를 사용하여 개발 후 EC2 배포
